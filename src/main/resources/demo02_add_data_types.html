Data Model

In order to make the data of the Wall of the Wise application more meaningful, we can utilize a simple HyperData design pattern: the hyperdata property. These properties allow to enrich simple untyped graphs with meaningful data types.

In simple terms, a hyperdata property consists of three nodes:

  • target node: this is the node for which the property is to be defined.
  • property value node: this node carries the value of the property.
  • property type node: this node represents the type of the property.
