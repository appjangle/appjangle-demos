Demo 05: Add Synchronization

While all posts in demo 04 are persisted using the Appjangle cloud storage, posts are not synchronized between multiple instances of the application.

In this demo we again make a few changes to the application so that posts made in one instance of the app are automatically synchronized with other instances of the app. Try opening multiple clients using the buttons below and see how changes are smoothly synchronized between all open clients!
