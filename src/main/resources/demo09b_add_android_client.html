Demo 09b: Add Mobile UI (Android)

In the previous demo, we have developed a Java Swing client for the Wall of the Wise. In this demo, we port this client to the Android environment.

You can install this demo from the Google Play Store and explore the full source code on GitHub.

Demo

Please follow the steps below to explore the Android demo:

1. Install App
Install 'Wall of the Wise' from Play Store
Download APK

You can either install the app from the Google Play store or download the APK and install it manually on your device or emulator.

2. Provide Wall URL

You need to open demo 05, demo 06, demo 07 or demo 08. After you have opened the client to post messages, copy the URL displayed on the bottom of the page.

Then, provide this URL in the Android application. Click on the item 'Custom URL' on the start page.

You can provide the URL of the wall in the pop up window and click [Submit].

Alternatively, you can also select one of the provided default wall URLs.

3. Observe Post Counts

The post counts for the wall will be displayed in a simple table on the Java application. Make more posts to the wall from where you have copied the URL to see the post counts in the Java app update automatically.

Key Components

The core components of the application are unchanged to the Java Swing Version.

However, instead of Swing panels, Android activities are used.
