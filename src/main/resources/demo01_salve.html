Demo 01: Salve!

In this first step of developing 'The Wall of the Wise', we define a simple data model to store wall posts on the Appjangle cloud and implement this model for a JavaScript and Java application.

Data Model

Appjangle is built on a hyperdata data model. Therefore, all data is expressed as nodes and (untyped) connections. Modelling our desired wall using this model is simple. Check the the Nextweb docs for details regarding the hyperdata modelling language.

A model for the Wall of the Wise could be defined as follows:

  • The wall itself can be represented by an arbitrary node 'posts'.
  • This wall is connected to one or more text nodes, which represent the posts.

Figure: Abstract data model

JavaScript

The following JavaScript application uploads some data according to the defined data model to the Appjangle cloud. Please note that the uploaded data satisfies both the abstract and concrete model defined above.

var session = Nextweb.createSession();

var posts = session.seed();

posts.append("Salve, Munde.");
posts.append("Salve, Nubes!");

session.commit().get(function(success) {
    console.log("Upload completed to: "+posts.get().uri());
    session.close().get(function(success) { });
});

Click the Run button to execute the application. You can click on the generated link to see the data uploaded to the Appjangle cloud.

You can also click the Edit button to make extensions and changes to the example, if you wish.

To learn more about the operations used in the example, check the Nextweb API documentation:

  • creareSession and close
  • seed
  • append
  • commit
Hello, World: Java

We can implement the same application in Java. You can copy the source code below it into any Java class. But please note that in order to run the following Java application, you need to download and link the Appjangle Java library from the downloads page. 

Session session = Nextweb.createSession();

Query posts = session.seed();

posts.append("Salve, Munde.");
posts.append("Salve, Nubes!");

session.commit().get();
System.out.println("Upload completed to: "+posts.get().uri());

session.close().get();
