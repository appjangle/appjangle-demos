Demo 01: Salve!

Objectives
  • Introduce hyperdata data modelling
  • Provide simple Hello, World! examples for Appjangle.
Requirements

While our new social network will without doubt have numerous unique and cool features, let's start with something simple for now:

On the wall of the wise, historical characters can leave inspiring and insightful comments to their heart's delight.

Requirement 1
Build a first, simple data model for The Wall of the Wise
Requirement 2
Upload a few example posts for a Wall of the Wise to the Appjangle cloud
Data Model

Appjangle is built on a hyperdata data model. Therefore, all data is expressed as nodes and (untyped) connections.

Modelling our desired wall using this model is no big deal. The wall itself can be represented by a node; posts to this wall are represented as text nodes, which are connected to the 'wall node'. For instance:

Figure: Naive data model

For more details regarding the data model as well as the modelling language used here, please refer to the Nextweb documentation! This first model is without doubt overly simplistic. But no worries, we will extend it later.

Hello, World: JavaScript

The following JavaScript application uploads some date according to the defined data model to the Appjangle cloud.

var session = Nextweb.createSession();

var posts = session.seed();

posts.append("Salve, Munde.");
posts.append("Salve, Nubes!");

session.commit().get(function(success) {
    console.log("Upload completed to: "+posts.get().uri());
    session.close().get(function(success) { });
});

Click the Run button to execute the application. You can click on the generated link to see the data uploaded to the Appjangle cloud.

You can also click the Edit button to make extensions and changes to the example, if you wish.

To learn more about the operations used in the example, check the Nextweb API documentation:

  • creareSession and close
  • seed
  • append
  • commit
Hello, World: Java

We can implement the same application in Java. You can copy the source code below it into any Java class. But please note that in order to run the following Java application, you need to download and link the Appjangle Java library from the downloads page. 

Session session = Nextweb.createSession();

Query posts = session.seed();

posts.append("Salve, Munde.");
posts.append("Salve, Nubes!");

session.commit().get();
System.out.println("Upload completed to: "+posts.get().uri());

session.close().get();
