The following pages feature a number of interactive demos, which showcase the capabilities of Appjangle. The demos describe a number of iterative development steps of in the development of a simple, yet highly innovative application:

We are making use of the Appjangle authorization system in order to separate shared and private data.

Demo 07: Add Java Client
