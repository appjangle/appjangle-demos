Summary

This demo allows to make posts to a status or message board. A text typed in the input box above will appear as item in a list below the input box upon clicking the [Post] button.

All posts are persisted to the Appjangle cloud storage. You can click on the button [Open in New Window] to try out loading the posts from the cloud.

Usage of Appjangle

Appjangle provides both a local embedded database for low latency access to the data as well as a cloud-based database to persist the post.

The usage of Appjangle is identical to the instant data example apart from the following alterations:

1. Creating a Cloud-based Seed node

In the first demo, the seed operation was used to create a seed node in an embedded database.

Before 

posts = session.seed("local");

For this demo, the additional parameter "local" is removed and the seed operation will default to request a seed node from the Appjangle cloud.

Now 

posts = session.seed();

2. That's IT

No further changes are required to turn the 'local' demo into an application with cloud-based persistence (apart from some JS wiring for reading hash codes).
